Setting up for development
--------------------------

Preparation work:
=================
- Install the site

```
shell> drush si emergency_site --db-url=mysql://root:root@localhost/humanintarian_response --site-name="Humanitarian Response" -y
```

- Download message library related modules

shell> cd sites/all/modules/
shell> git clone git@github.com:Wiredcraft/message_library.git message_library
shell> cd message_library


1. Enable features


2. Prepare the taxonomies

```
drush prepopulate-ml-terms
```

3. Prepare some content for testing
```
drush

```

4.Make sure solr is configured and working propertly

- Make essential configuration at: `admin/config/search/search_api`

  * Solr host: localhost
  * Solr port: 8983
